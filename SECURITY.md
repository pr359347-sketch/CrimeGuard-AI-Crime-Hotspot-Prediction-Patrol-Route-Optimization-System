## Reporting a Vulnerability

If you discover a security vulnerability, please contact:

Email:
security@crimeguard-ai.tech

Please include:

- Description
- Steps to reproduce
- Expected impact

A response will be provided within 7 business days.

Thank you for helping improve the security of CrimeGuard-AI.
